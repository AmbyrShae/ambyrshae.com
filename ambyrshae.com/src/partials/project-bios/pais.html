Public Assistance Information System

Providing a digital solution for applying to SNAP / TANF benefits in order to promptly receive assistance.

Learn more at https://pais-benefits.dhs.hawaii.gov/

Before the Covid-19 pandemic, Hawaii SNAP / TANF applications were done on paper. When offices shutdown, it created a bottleneck for those who needed assistance. I designed the online application. Within a day of the site going live, there were 1000 applications in progress. My main design principles were to help users easily navigate and complete their applications. This included single column layout which was responsive to mobile devices and using bold colors for call to action elements. Designs were done using Adobe XD.
